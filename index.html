Student Management System

C Language Console-Based Project

üìå Project Introduction

This Student Management System is a console-based application developed using the C programming language. The project is created to understand how student records can be stored, accessed, updated, and deleted using basic programming logic.

The entire system runs in the terminal and does not use any external libraries or databases. All operations are handled using core C concepts.

üß† What This Program Actually Does

  • Stores student information such as roll number, name, and marks
  • Allows the user to add multiple student records
  • Displays all stored student records
  • Searches a student using roll number
  • Updates existing student data
  • Deletes student records when required
  • Calculates total marks and percentage

  • The program uses menu-driven logic with switch-case
  • Each menu option calls a separate function
  • Student data is written to a file using file handling
  • Data remains saved even after program termination
  • Loops are used to repeat operations until the user exits

üíæ File Handling Used

This project uses file handling to store data permanently.

  • fopen() to open files
  • fwrite() to store student records
  • fread() to read stored data
  • fclose() to close the file

Because of this, student data is not lost when the program is closed.
